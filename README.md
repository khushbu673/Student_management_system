# Student Management System

A lightweight Java web app to manage student data using Servlets, JSP, JDBC, and MySQL.

## Features
- Add, View, Edit, Delete students (CRUD)
- Login page
- MySQL database backend

## Setup
1. Run `database.sql` in MySQL
2. Update DB credentials in `DBconnection.java`
3. Add MySQL Connector/J to build path
4. Deploy on Apache Tomcat
5. Open `http://localhost:8080/StudentManagementSystem/Index.jsp`

## Tech Stack
Java | JSP | JDBC | MySQL | Apache Tomcat
